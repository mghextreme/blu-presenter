# Blu Presenter

## App

Built around [react-router-dom](https://reactrouter.com/).

```sh
cd app
```

### Install

```sh
pnpm install
```

### Develop

```sh
pnpm dev
```

### Build

```sh
pnpm build
```

## API

Built around [NestJS](https://nestjs.com/).

```sh
cd api
```

### Install

```sh
pnpm install
```

### Develop

```sh
pnpm start:dev
```

### Build

```sh
pnpm build
```
