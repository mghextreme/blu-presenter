# BluPresenter

[Official website](https://www.blupresenter.com)

## Supabase

Check [Supabase documentation](https://supabase.com/docs).

### Development

```sh
supabase start
```

The application seeding will add one user:

- Email: `admin@example.com`
- Password: `password`

## App

Built around [react-router-dom](https://reactrouter.com/).

```sh
cd app
```

### Install

```sh
pnpm install
```

### Develop

```sh
pnpm dev
```

### Test

```sh
pnpm test run
```

### Build

```sh
pnpm build
```

## API

Built around [NestJS](https://nestjs.com/).

```sh
cd api
```

### Install

```sh
pnpm install
```

### Develop

```sh
pnpm start:dev
```

### Test

```sh
pnpm test
```

### Build

```sh
pnpm build
```
